## get/post请求

通过装饰器来确定请求类型

```python
from fastapi import FastAPI

app = FastAPI()

# get请求
@app.get("/")
def get_name():
    return 'hello world!'


```

## 请求参数

### query参数

```python
# 例如：http://localhost:8080/item?id=123
@app.get('/item')
def get_item(id:str):
    return id
```

其中的id就是query参数

### 路径参数

```python

# http://localhost:8000/item/3/
@app.get("/item/{item_id}")
def get_item(item_id:int):
    return item_id
```
### 请求体

```python

class Item:
    name: str
    description: str|None

@app.post("/item/new")
def add_new_item(item:Item):
    return item
```
