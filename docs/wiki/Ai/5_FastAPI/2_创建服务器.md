## 添加依赖

```bash
uv add fastapi
```

## 创建服务器

### 开发阶段

使用``uvicorn``

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

- reload: 检测到文件修改自动加载
- main.app: 指定main.py中的app实例
  - ``app = FastApi()po``

或者

```python
if __name__ == '__main__':
    uvicorn.run("main:app",host='127.0.0.1',port=8000,reload=True)
```
