## 响应类型

通过注解函数的返回类型来声明响应类型。

```python
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class Item(BaseModel):
    name: str

@app.get('/item')
def get_item(name:str) -> Item:
    return Item(name="str")
```

## 响应模型

通过`response_model`控制

> `response_mode = None`会禁用输出响应模型

```python
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class InputModel(BaseModel):
    name: str
    password: str

# 输出过滤掉密码
class OutPutModel(BaseModel):
    name: str

# 禁用响应
@app.post("/newUser", response_model= None)
def add_new_user(user: InputModel):
    return user

# 实际会过滤掉其中的password字段
@app.post("/newUser", response_model= OutPutModel)
def add_new_user(user: InputModel):
    return user
```
