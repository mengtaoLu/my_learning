from typing import Any

## 摘要和描述

可以使用`summary`和`description`

```python
from fastapi import FastAPI
from typing import Any

app = FastAPI()

@app.post("/item",summary="create item"
    ,description="create an item with name and description")
def create_item(item: Any):
    return item
```

## 文档字符串描述

如果描述过长的时候，`fastapi`可以读取文档描述字符串

```python
from fastapi import FastAPI

app = FastAPI()

@app.post("/item",summary="create an item")
def create_item(item:str | None):
    """
    create an item
    ...
    ...
    """
    return item
```

## 描述响应 `response_description`

```python
from fastapi import FastAPI

app = FastAPI()

@app.post("/item",response_description="The created item")
def create_item(item:str | None):
    return item
```
