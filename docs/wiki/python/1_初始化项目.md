使用`uv`初始化项目。

## 安装`uv`

```bash
brew install uv
```

## 初始化

```bash
mkdir my-dir
cd ./my-dir

uv init

# 创建一个虚拟环境.venv
uv venv
```

## 激活虚拟环境

### 传统方式：命令行激活

```bash
source .venv/bin/activate
```

### 现代方式：UV激活

```bash
uv run python main.py
```

## 其他技巧

### 指定python版本

```bash
uv python install 3.12
uv init --python 3.12
```

### 添加依赖

```bash
uv add fastapi

# 添加多个依赖
uv add requests python-dotenv
```

#### 锁定版本依赖

```bash
uv sync
```
