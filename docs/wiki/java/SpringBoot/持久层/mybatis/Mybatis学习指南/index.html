MyBatis 简介

MyBatis 是一款优秀的持久层框架，它支持自定义 SQL、存储过程以及高级映射。

关键知识点

  • 1半自动 ORM：相比 JPA 的全自动，MyBatis 需要手写 SQL，换取更高的灵活性。
  • 2消除 JDBC 冗余：自动处理参数设置和结果集封装。
  • 3SQL 与代码解耦：通过 XML 或注解将 SQL 逻辑从 Java 代码中分离。
  • 4简单易学：基于原生 SQL，只要会 SQL 就能上手。
MyBatis 核心架构
SqlSessionFactoryBuilder -> SqlSessionFactory -> SqlSession -> Executor -> Mapped Statement

进阶提示

掌握了本节内容后，建议通过实践一个简单的 CRUD 项目来巩固。你可以结合之前的 OrbStack 部署 PostgreSQL 来测试你的 SQL 映射！
