## @RequestParam

针对 GET 请求，获取 URL 中的参数

`/api/test?name=abc`

```java
    @GetMapping("/api/test")
    public String test2(@RequestParam String name) {
        return name;
    }
```

## @PathVariable

获取 URL 中的动态参数

`/test/{id}`

```java
    @GetMapping("/test/{id}")
    public String test3(@PathVariable("id") String id) {
        return id;
    }
```

## @RequestBody

获取 POST 方法中的 body

POST `/register`

```java
    @PostMapping("/register")
    public String register(@RequestBody Map<String,Object> userInfo) {
        return userInfo.get("name").toString();
    }
```
